# Dental Sales Performance Dashboard

Interactive sales dashboard built using Microsoft Excel to analyze sales performance, customer activity, and operational reporting.

## Tools Used
- Microsoft Excel
- Pivot Table
- Pivot Chart
- XLOOKUP
- Slicers
- Data Cleaning

## Dashboard Features
- Sales Performance Analysis
- Revenue Monitoring
- Customer Activity Tracking
- KPI Reporting
- Interactive Dashboard Filters

## Key Insights
- Identified top-performing products and sales trends.
- Analyzed customer purchasing activity.
- Improved reporting readability using dashboard visualization.
